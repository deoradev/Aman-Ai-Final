# Security Policy

## Supported Versions

We are committed to ensuring the security of our application. Please use the latest version to receive security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

We take all security vulnerabilities seriously. If you believe you have found a security vulnerability in Aman Digital Care, please report it to us through one of the following channels:

- **Email**: Send an email to `officialamanfoundation@gmail.com` with the subject line "Security Vulnerability Report".

Please include the following details in your report:
- A description of the vulnerability and its potential impact.
- Steps to reproduce the vulnerability.
- Any proof-of-concept code or screenshots.

We will acknowledge your email within 48 hours and will work to resolve the issue as quickly as possible. We kindly ask that you do not publicly disclose the vulnerability until we have had a chance to address it.
